# `seksbot tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
seksbot tui
seksbot tui --url ws://127.0.0.1:18789 --token <token>
seksbot tui --session main --deliver
```
