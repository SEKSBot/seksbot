# `seksbot setup`

Initialize `~/.seksbot/seksbot.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
seksbot setup
seksbot setup --workspace ~/.seksbot/workspace
```

To run the wizard via setup:

```bash
seksbot setup --wizard
```
